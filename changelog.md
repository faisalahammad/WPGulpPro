### RELEASE: [4.0.0](https://github.com/faisalahammad/WPGulp/compare/3.2.0...4.0.0)

**Node 20/22/24 LTS Compatibility Release**

![📦 NEW:](https://img.shields.io/badge/-NEW-gray.svg?colorB=3778FF)

> 📦 Renamed to `wpgulppro` [`4.0.0`](https://github.com/faisalahammad/WPGulp) <br>
> 📦 Dart Sass support via `sass` package [`package.json`](https://github.com/faisalahammad/WPGulp) <br>
> 📦 `.nvmrc` file with `lts/*` [`nvmrc`](https://github.com/faisalahammad/WPGulp) <br>

![👌 IMPROVE:](https://img.shields.io/badge/-IMPROVEMENT-gray.svg?colorB=39AA54)

> 👌 Node engine requirement to `>=20.0.0` [`package.json`](https://github.com/faisalahammad/WPGulp) <br>
> 👌 Installer dependencies to CJS-compatible versions (chalk v4, execa v5, ora v5) [`package.json`](https://github.com/faisalahammad/WPGulp) <br>
> 👌 Template dependencies to latest compatible versions (gulp-sass v5, browser-sync v2.29, etc.) [`package.json`](https://github.com/faisalahammad/WPGulp) <br>
> 👌 gulp-sass to use Dart Sass [`gulpfile.babel.js`](https://github.com/faisalahammad/WPGulp) <br>
> 👌 GitHub URLs to faisalahammad/WPGulp [`installer/index.js`](https://github.com/faisalahammad/WPGulp) <br>
> 👌 Async/await pattern in installer [`installer/index.js`](https://github.com/faisalahammad/WPGulp) <br>

![🐛 FIX:](https://img.shields.io/badge/-FIX-gray.svg?colorB=ff6347)

> 🐛 Utility files had swapped contents [`handleError.js`, `printNextSteps.js`](https://github.com/faisalahammad/WPGulp) <br>

![📖 DOC:](https://img.shields.io/badge/-DOCS-gray.svg?colorB=978CD4)

> 📖 Updated readme.md for WPGulpPro [`readme.md`](https://github.com/faisalahammad/WPGulp) <br>
> 📖 Added AUDIT.md with full dependency analysis [`AUDIT.md`](https://github.com/faisalahammad/WPGulp) <br>

![‼️ BREAKING:](https://img.shields.io/badge/-BREAKING-gray.svg?colorB=D94093)

> ‼️ Requires Node 20 or higher [`engines`](https://github.com/faisalahammad/WPGulp) <br>
> ‼️ Package renamed to `wpgulppro` - use `npx wpgulppro` instead of `npx wpgulp` <br>
> ‼️ Uses Dart Sass - node-sass users must migrate [`sass`](https://github.com/faisalahammad/WPGulp) <br>
